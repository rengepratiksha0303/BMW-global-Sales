# BMW Global Sales Prediction 🚗

This project predicts BMW car sales using Machine Learning and Deep Learning models.

Models Used:
- KNN (K-Nearest Neighbors)
- ANN (Artificial Neural Network)
- CNN (Convolutional Neural Network)

Dataset:
BMW Global Sales Dataset (2018–2025)

Technologies:
- Python
- Scikit-learn
- TensorFlow / Keras
- Streamlit
